# SES SERVER

## Installation

To install the dependencies, run the following command:

```bash
npm install
```

## Usage

To start the server, run the following command:

```bash
node index.js
```

## Dependencies

- @aws-sdk/client-ses: ^3.929.0
- aws-sdk: ^2.1692.0
- dotenv: ^17.2.3
- express: ^5.1.0
